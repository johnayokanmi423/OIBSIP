# Data Analytics & Machine Learning Projects

## Overview

This repository contains a collection of data analytics and machine learning tasks completed as part of hands-on practice and applied learning. Each notebook focuses on solving a specific business or analytical problem using Python, statistical analysis, and machine learning techniques.

The projects emphasize data cleaning, exploratory data analysis, feature engineering, modeling, and interpretation of results rather than just code execution.



##  Project Contents

To segment customers into distinct groups based on their behavior and attributes in order to support targeted marketing, personalization, and business decision-making.
